1.  cd stendur fyrir change directory
	ls býr til lista með öllum gögnum in í directoríinu
	pwd stendur fyrir present working directory og það prentar út nafnið á directoríinu sem þú ert nuna að nota
2.  git clone copiar repositoríin inn í nýa möppu
	git status sýnir statusið á möppuna sem er tengd repositoríið og sýnir hvað vantar að commita
	git diff sýnir munin milli commitar í repository
3.  git log sýnir öll commitin 
	updatar filinn sem í directoríin sem þú ert í til að vera alveg eins og það sem þú valdir að herma eftir
	git branch leyfir þig að breyta branchið sem þú valdir eins og þú vilt t.d. create, búa til lista með branchinu, deleta branch og mikið fleirra
4.  útgáfustýring er eins og save files í tölvu leiki. það er notað þannig ef þú geri 
	stóra villu þú getur farið til baka í síðasta útgáfa
5.  það eru margar kostir við að nota git t.d. ef þú ert að gera verkefni með fleyrri 
	fólk eða tölvur.
6.  eins mikið og þarf að commita
7.  working directory er directoríið sem þú ert buin að tengja við cmd eða git bash( og 	líka fleirra auðvitað) til að nota commands eins og mkdir og svo fleirra.
